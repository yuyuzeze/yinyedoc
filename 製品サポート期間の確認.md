## サポート終了が想定される製品への対応方針

| No. | 対象製品 | 想定時期 | 後継製品への更新計画 |
|-----|--------|--------|-----------------|
| 1 | Angular 20 | 2026年11月（LTS 終了） | LTS 終了前に次期 LTS バージョン（Angular 22 等）への更新を計画する。Angular CLI の ng update コマンドにより段階的にアップグレードが可能である |
| 2 | .NET 10 | 2028年11月（LTS 終了） | LTS 終了前に次期 LTS バージョン（.NET 12 等）への更新を計画する。TargetFramework および NuGet パッケージの更新が主な作業となる |
| 3 | ExcelCreator / ActiveReports | 保守契約終了時 | 保守契約を継続するか、後継バージョンへの更新を判断する。更新にはライセンス費用が発生する |
| 4 | Windows Server 2025 | 2029年10月（Mainstream 終了） | Extended サポート（2034年10月まで）への移行、または次期 OS バージョンへの更新を計画する |

## 備考

- Azure マネージドサービス（Azure SQL Database、Entra ID、Blob Storage、Key Vault 等）は、Microsoft が自動的にパッチ適用・バージョン更新を行うため、個別のサポート終了日は設定されない。
- 商用ライブラリ（ExcelCreator、ActiveReports）については、有償の保守契約・サービス契約の継続によりサポート及びセキュリティパッチの提供が維持される。契約更新にかかる費用は本調達に含むものとする。
- サポートライフサイクルポリシーの確認にあたっては、政府情報システムにおけるサポート終了等技術への対応に関する技術レポート等を参考にする。
- サポート終了が想定される場合は、後継製品への更新計画を事前に提出する。後継製品に更新する場合の費用は本調達に含むものとする。
