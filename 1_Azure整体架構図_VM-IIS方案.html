📋 数据流总结 (Data Flow Summary)
🔵 用户访问流
用户 → 浏览器
→ VPN 加密隧道
→ VPN Gateway
→ NSG (Web) 防火墙
→ IIS (/ → Angular SPA)
→ HttpClient → /api
→ IIS → Kestrel → .NET 8
→ EF Core → Azure SQL
🟠 文件交换流
用户上传：
Angular → API → Blob Storage

外部系统交换：
① SFTP → VM → FileWatcher → Blob
② 外部系统 → SAS Token → Blob
③ API → SSH.NET → 外部 SFTP
🔴 外部 API 通信流
API 调用其他系统：
.NET 8 HttpClient → Polly (重试)
→ NSG Outbound → 外部 API

其他系统调用本系统：
外部 → VPN → NSG → IIS /api
→ JWT 验证 → Controller 处理
